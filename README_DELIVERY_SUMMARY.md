## ✏️ How to Edit Content

### Change Newsletter Text

All content is in: **`app/page.tsx`**

```typescript
// Example: Change the main title
<h1 className={styles.heroTitle}>New Title Here</h1>

// Example: Update the date
<span className={styles.date}>July 2026</span>

// Example: Change contact email
<a href="mailto:newemail@freshriver.ai">newemail@freshriver.ai</a>
```

### Publish Changes

```bash
git add app/page.tsx
git commit -m "Update: [what you changed]"
git push origin main
# Vercel auto-deploys in 1-2 minutes
```

## 📞 Support & Troubleshooting

### Most Common Issues & Fixes

**"Git authentication failed"**
→ Use Personal Access Token from https://github.com/settings/tokens

**"Can't find my repo in Vercel"**
→ Make sure code is pushed to GitHub first

**"Vercel build failed"**
→ Check Deployments tab in Vercel for error logs

**"Site looks broken on mobile"**
→ Hard refresh: Ctrl+Shift+R or Cmd+Shift+R

**"How do I change the content?"**
→ Edit `app/page.tsx` and push to GitHub

**"Can I use a custom domain?"**
→ Yes, add it in Vercel project settings
